# FlowForge Cozy UI Theme

## Brand Feel

FlowForge should feel like a calm, premium workspace for focused knowledge work. The visual tone is modern SaaS with cues from Linear, Notion, Raycast, and Vercel: restrained, crisp, useful, and quietly warm.

Avoid bright, cartoonish, glossy, or heavily saturated UI treatments. Color should guide attention without overwhelming the workspace.

## Color Palette

| Role | Color | Usage |
| --- | --- | --- |
| Background | `#0F172A` | App canvas and main page background |
| Sidebar | `#111827` | Persistent navigation surface |
| Card | `#1F2937` | Panels, previews, and dashboard cards |
| Border | `#334155` | Subtle separation between surfaces |
| Primary | `#6366F1` | Primary actions, selected states, brand accents |
| Secondary Accent | `#A78BFA` | AI, assistant, and highlight details |
| Text Primary | `#F8FAFC` | Main readable text |
| Text Secondary | `#CBD5E1` | Supporting labels and metadata |
| Text Muted | `#94A3B8` | Hints, captions, inactive states |

## Module Accents

Use muted icon colors to distinguish modules:

- Dashboard: Indigo
- AI Assistant: Purple
- Calendar: Blue
- Task / Kanban: Orange
- Notes: Emerald
- Whiteboard: Pink
- Pages / Spaces: Cyan
- AI Template Builder: Violet
- Settings: Slate gray

## Typography

- Use the default system sans stack for a crisp SaaS feel.
- Prefer medium and semibold weights over heavy display weights.
- Use small uppercase labels sparingly for groups and metadata.
- Keep dashboard and panel copy short, direct, and operational.

## Spacing And Shape

- Use generous page spacing with compact navigation density.
- Use `rounded-xl` for major controls, cards, sidebar rows, and panels.
- Avoid nested cards unless the inner element is a true repeated item or control.
- Use stable dimensions for navigation rows, icon buttons, and preview cards.

## Surfaces And Motion

- Layer slate surfaces with subtle borders and low-opacity fills.
- Use soft shadows only to create depth on major panels.
- Hover states should be smooth, quiet, and slightly brighter than rest states.
- Avoid loud gradients, decorative blobs, and unnecessary illustration.

## Icon Guidelines

- Use Lucide icons throughout the app.
- Keep sidebar icons colorful but muted.
- When navigation is collapsed, preserve icon visibility and provide accessible labels.
- Prefer familiar icons over text-heavy controls for compact utility actions.
